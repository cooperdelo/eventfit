# Image Assets

Place your images in the following directories:

## Directory Structure

```
public/images/
├── hero/
│   ├── gameday.jpg
│   ├── formal.jpg
│   ├── concert.jpg
│   └── date-night.jpg
├── how-it-works/
│   ├── calendar-preview.jpg
│   ├── outfit-board-preview.jpg
│   └── rental-preview.jpg
└── about/
    └── diverse-students.jpg
```

## Image Requirements

### Hero Images (`/images/hero/`)

- **Size**: 600x800px (3:4 aspect ratio)
- **Format**: JPG or WebP
- **Content**: Real students in outfits for each event type
- **Files**:
  - `gameday.jpg` - Gameday/tailgate outfit
  - `formal.jpg` - Formal event outfit
  - `concert.jpg` - Concert/nightlife outfit
  - `date-night.jpg` - Date night outfit

### How It Works Images (`/images/how-it-works/`)

- **Size**: 1200x675px (16:9 aspect ratio)
- **Format**: JPG or WebP
- **Content**: Screenshots or mockups of the app
- **Files**:
  - `calendar-preview.jpg` - Calendar view showing events
  - `outfit-board-preview.jpg` - Outfit feed/grid view
  - `rental-preview.jpg` - Rental confirmation screen

### About Page Images (`/images/about/`)

- **Size**: 800x800px (1:1 aspect ratio)
- **Format**: JPG or WebP
- **Content**: Diverse group of college students
- **Files**:
  - `diverse-students.jpg` - Group photo of diverse students

## Notes

- All images should be optimized for web (compressed, appropriate file size)
- Use WebP format when possible for better performance
- Images will display with blur placeholders until loaded
- Ensure all images have proper alt text for accessibility
